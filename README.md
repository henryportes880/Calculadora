# 🧮 Calculadora Modular em Node.js

Este projeto é uma calculadora simples desenvolvida em **Node.js**, com o objetivo de demonstrar o conceito de **modularização**, ou seja, a separação do código em diferentes arquivos para facilitar a manutenção, organização e reutilização.

---

## 🚀 Funcionalidades

A aplicação realiza as quatro operações matemáticas básicas:

- ➕ Soma  
- ➖ Subtração  
- ✖️ Multiplicação  
- ➗ Divisão  

---

## 📂 Estrutura do Projeto

    📁 projeto-calculadora
    ├── calcModulo.js      # Arquivo principal (menu e interação com o usuário)
    ├── soma.js            # Módulo de soma
    ├── subtracao.js       # Módulo de subtração
    ├── multiplicacao.js   # Módulo de multiplicação
    ├── divisao.js         # Módulo de divisão
    └── package.json       # Gerenciador de dependências

---

## 🛠️ Tecnologias Utilizadas

- Node.js  
- prompt-sync (entrada de dados no terminal)

---

## 🔧 Como Instalar e Executar

### 1. Pré-requisitos

Certifique-se de ter o **Node.js** instalado na sua máquina.

### 2. Instalação

    npm install

### 3. Execução

    node calcModulo.js

---

## 📚 Objetivo do Projeto

Este projeto foi desenvolvido para fins educacionais, com foco em:

- Praticar modularização em Node.js  
- Organizar código em múltiplos arquivos  
- Melhorar a legibilidade e manutenção
